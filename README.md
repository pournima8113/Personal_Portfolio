# 🚀 Personal Portfolio | Pournima Akole

Welcome to my professional portfolio repository! This project showcases my skills, education, and projects as a **Computer Engineer**, **Front-end Developer**, and **Data Analyst**.

## 🌐 Live Demo
You can view the live website here: https://pournimaapersonalportfolio.netlify.app/


## ✨ Features
- **Responsive Design:** Mobile-friendly and optimized for all screen sizes.
- **Dark/Light Mode:** Interactive toggle for a personalized viewing experience.
- **Skill Sections:** Showcasing expertise in Web Development, Data Analysis, and Programming.
- **Direct Download:** Integrated "Download CV" button for recruiters.
- **Contact Form:** Functional contact section powered by Formspree.


## 🛠️ Tech Stack
- **Languages:** HTML5, CSS3, JavaScript
- **Frameworks/Libraries:** Bootstrap, FontAwesome
- **Tools:** VS Code, Git, GitHub


## 📁 Project Structure
- `index.html`: The main structure of the website.
- `style.css`: All the styling, including Dark Mode and Responsive layouts.
- `script.js`: Functionality for the Dark Mode toggle and Mobile Menu.
- `assets/`: Folder containing certificates and resume PDF.
